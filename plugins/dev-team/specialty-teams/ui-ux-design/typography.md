## Worker Focus
Platform system font (SF Pro, Roboto, Segoe UI Variable, system-ui), body 14-17pt (16px default), minimum 11-12pt for captions, line height 1.4x-1.5x, 2-3 font weights per screen, 45-75 character paragraph width

## Verify
System font used (not custom fonts without justification); body text ≥14pt; captions ≥11pt; no more than 3 weights per screen; no all-caps for more than a few words
