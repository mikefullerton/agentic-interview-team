## Worker Focus
Use built-in WinUI 3 controls — they implement Fluent 2 natively; never custom-draw what a standard control can do; use Segoe UI Variable for typography; use Segoe Fluent Icons for iconography; follow Windows design guidance for layout, spacing, and navigation patterns

## Verify
No custom-drawn controls where standard WinUI 3 equivalents exist; Segoe UI Variable used for text; Segoe Fluent Icons used for icons; layout follows Windows spacing and navigation patterns
