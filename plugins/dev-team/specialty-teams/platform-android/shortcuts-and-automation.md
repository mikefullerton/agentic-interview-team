## Worker Focus
Use `AppActions` for Google Assistant integration; support `Intent`-based automation; expose key app actions as App Actions with `actions.xml`; use Shortcuts API for pinned and dynamic shortcuts

## Verify
`actions.xml` present with App Actions declarations; key app flows reachable via Google Assistant; dynamic shortcuts created for frequently used actions; shortcuts updated on state changes
