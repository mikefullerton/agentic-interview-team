## Worker Focus
CSS Grid/Flexbox with responsive breakpoints via media queries (no hard-coded widths); single-column default expanding to multi-column; no nested same-direction scroll; content-first element order

## Verify
No inline style pixel widths for layout; breakpoints use `em`/`rem` or device-class media queries; no nested overflow-y in overflow-y containers; DOM order matches visual reading order
