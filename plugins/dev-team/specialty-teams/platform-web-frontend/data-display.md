Implement correct HTML patterns — `<ul>/<li>` for lists, `<table>` with sortable columns for tabular data, card grid for heterogeneous browsable content; wire up sort/filter at 10+ items, search at 50+

## Verify
Tables rendered with `<th>` and sort controls; lists use semantic HTML; collections ≥10 have sort or filter; collections ≥50 have search input
