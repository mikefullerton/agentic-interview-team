## Worker Focus
Feedback weight matches action weight — inline for fields, toast for non-critical confirmations (3-5s auto-dismiss), banner for persistent warnings, dialog only for destructive/irreversible; no dialogs for success

## Verify
No success messages in dialogs; destructive actions require dialog with explicitly labeled action (not "OK"); default focus on safe option (Cancel); toast auto-dismisses 3-5s
