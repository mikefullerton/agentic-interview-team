## Worker Focus
System font stack, body 16px default, minimum 11-12px captions, line-height 1.4-1.5x, 2-3 weights per screen, paragraph max-width 45-75ch, `prefers-reduced-data` reduces font loading

## Verify
System font stack in CSS; body `font-size` ≥14px; no element `font-size` <11px; `line-height` ≥1.4 on body text; paragraph `max-width` ≤75ch
