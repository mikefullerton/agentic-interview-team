## Worker Focus
When no native solution exists, research battle-tested open-source libraries and present options before building custom; custom implementation is a deliberate choice not a default; evaluate options on maintenance health, license, and fit

## Verify
Custom implementations of solved problems (parsers, serializers, HTTP clients, caches) are justified; open-source alternatives were evaluated; chosen library has an active maintenance history and compatible license
