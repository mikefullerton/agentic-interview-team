## Worker Focus
Windows does not have a native Handoff equivalent — use deep links as universal handoff payload; protocol activation URIs as cross-device handoff mechanism; fall back gracefully when receiving device lacks a feature; web-resolvable URLs for cross-platform continuity

## Verify
Key views have protocol activation URIs suitable for cross-device handoff; URLs resolve on web or mobile equivalents; state encoded in URI is sufficient to restore context
