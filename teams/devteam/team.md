# Dev Team

## Role

Full-stack product analysis and development consulting. Provides specialist review across all technical domains — architecture, platforms, security, testing, accessibility, UI/UX, code quality, and more.

## Specialists

Each specialist owns a domain and dispatches specialities (focused review units) within that domain. See `specialists/<name>/specialist.md` for each specialist's role, persona, and speciality manifest.

## Consulting

Cross-cutting consulting teams provide multi-specialist analysis on topics that span domain boundaries. See `consulting/<area>/` for each consulting team's concerns.

## Navigation

Every directory in this team has an `index.md` that serves as a table of contents for that directory. Start at `index.md` in any directory to discover what's inside.
