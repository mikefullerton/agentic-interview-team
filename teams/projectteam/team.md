# Project Team

## Role

Project management — schedules, milestones, task breakdown, issue tracking, dependency mapping, risk identification, decision recording, and project status oversight.

## Specialists

- **project-manager** — Tracks what matters, flags what's drifting, and never lets a decision go unrecorded. See `specialists/project-manager/specialist.md`.

## Specialities

The project-manager specialist dispatches six specialities: schedule, todos, issues, concerns, dependencies, and decisions. Each corresponds to a distinct project data resource.

## Navigation

Every directory in this team has an `index.md` that serves as a table of contents for that directory. Start at `index.md` in any directory to discover what's inside.
